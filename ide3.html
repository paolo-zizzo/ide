Notepad++ est un éditeur de texte libre générique,
fonctionnant sous Windows,
codé en C++,
qui intègre la coloration syntaxique de code source
pour les langages et fichiers C,

Ce logiciel, fondé sur la composante Scintilla,
a pour but de fournir un éditeur légeret efficace.
Il est également une alternative au bloc-notes de Windows!
Le projet est sous licence GPL version 2.
Un équivalent sous Linux serait Gedit ou encore Kate.
